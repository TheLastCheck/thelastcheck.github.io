Check MICR Edit and Simulated Sort Program

The Check MICR Edit and Simulated Sort program provides editing capabilities for Java programs such as Vector Image Exchange (VIEX). This removes any requirement for SCI programs or other mainframe- based edit definition programs. It also removes any requirement for CPCS or SuperMICR for creating or maintaining the editing process.

The TLC Check MICR Edit and Simulated Sort program uses its own simple to use, and easy to extend, scripting language for defining the edits and algorithms to be performed on the check items. This scripting language is dynamically compiled into Java byte-code and runs in the same JVM as the calling program instead of in a separate process space. Since it is actual Java byte-code, it runs much faster than the older SCI 3890 emulators.
